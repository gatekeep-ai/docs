# Getting Your API Key

To get your API key from [Gatekeep Platform](https://platform.gatekeep.ai/keys), follow these steps:

1. Sign up for an account on Gatekeep Platform if you haven't already.
2. Set up a billing account to access the API keys.
3. Navigate to [API Key's Page](https://platform.gatekeep.ai/keys) in your web browser.
4. Copy your **API key** from the page. **YOU WILL NOT BE ABLE TO SEE IT AGAIN**
